Python Quiz Game

A simple command-line Python Quiz Game that tests your Python programming knowledge with multiple-choice questions. This project is great for beginners who want to practice Python concepts while improving their programming skills.

 Features:
 -10 Python multiple-choice questions
 -Instant feedback for each answer
 -Final score displayed at the end
 -Easy to add or modify questions
 -Questions stored using Python dictionaries
 -Runs directly in the terminal
